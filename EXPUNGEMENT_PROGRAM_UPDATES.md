## For State Funding Documentation:

All program claims are now either:
1. **Aspirational** (timeline estimates)
2. **Geographic fact** (88 counties)
3. **Service description** (what's included)
4. **Pricing transparency** (sliding scale)
5. **Status** (launching 2026, accepting applications)

No performance metrics are claimed until you have actual data to support them.
